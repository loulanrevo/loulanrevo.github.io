📝 The Complete Guide to Code Display & File Downloads

This article demonstrates how to elegantly display code in your blog and provide file download functionality.

🔧 Tech Stack & Plugins

Layer Tech/Plugin Description
Blog Framework Hexo v8.1.1 Static blog generator
Theme Butterfly v5.5.4 Provides code highlighting, Mac-style code blocks, copy button
Code Highlighting PrismJS Theme darker, supports auto-wrap & height limit
Custom Tag download-tag.js Custom Hexo Tag plugin, provides codefile & filecard tags
Style Injection darkmode.css Custom download button & card styles, supports light/dark theme switching

1. Basic Code Block (Built-in Copy Button)

The Butterfly theme already has built-in code copy functionality. Just use standard Markdown syntax:

Hover over the top-right corner of the code block, and a copy button will appear —click it to copy the entire code.

2. Code Block with Download Button

If you want readers to directly download a .py file, use the custom download tag:

fibonacci.py

Download File

Syntax Reference:

Click the Download File button above, and the browser will directly download the fibonacci.py file.

3. File Download Card

If you want to provide a download link for an external file, use the filecard tag:

🐍 Data Analysis Script.py Python · 12.5 KB Download

Syntax Reference:

This card automatically displays the corresponding icon based on the file extension (e.g., 🐍 for Python).

6. Usage Recommendations

Scenario Recommended Approach Reason
Short code snippets Regular code block Simple, built-in copy function
Complete runnable scripts codefile tag Readers can download and run directly
External resources / large files filecard tag Aesthetic, shows file info
Multi-file packages filecard + ZIP Convenient batch download

7. Configuration Notes

The following configurations have been set up for you:

  • Code highlighting theme: Switched to darker dark theme, better suited for technical content
  • Mac-style code block: Enabled rounded corners and title bar style
  • Code auto-wrap: Avoids horizontal scrollbar
  • Code height limit: Shows expand button when exceeding 500px
  • Download button styles: Injected into darkmode.css, supports light/dark theme switching

No additional setup needed —just use the syntax above in your posts!
